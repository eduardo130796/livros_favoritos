Neste sistema o usuário pode pesquisar livros, marcar os que gostou como favoritos e escrever suas anotações.

1. O que o sistema faz:

    - Permite buscar livros usando a API pública do Google Books.

    - Mostra uma lista com os resultados da busca.

    - O usuário pode favoritar um livro.

    - Na página de favoritos, é possível:

        - Ver todos os livros salvos.

        - Escrever anotações sobre cada livro.

        - Salvar várias anotações.

        - Excluir anotações.

        - Excluir o livro dos favoritos.

2. Página Inicial – Busca de Livros

    - O usuário digita o nome de um livro.

    - O sistema usa a API do Google Books para buscar os resultados.

    - Cada livro encontrado aparece com:

        - imagem

        - título

        - botão Favoritar


3. Página de Favoritos

    - Aqui aparecem todos os livros que o usuário salvou.

    - Para cada livro é possível:

        - Escrever anotações

        - Salvar quantas anotações quiser

        - Remover anotações individualmente

        - Excluir o livro dos favoritos



4.Resumo do Funcionamento:

    1. O usuário busca um livro pela API.

    2. Escolhe quais quer salvar como favoritos.

    3. Entra na página de favoritos.

    4. Escreve anotações sobre cada livro.

    5. Pode excluir anotações e livros quando quiser.

    6. Todos os dados ficam guardados no navegador.
